# Grade 3 English Comprehension Exams

This folder contains adapted and expanded Grade 3 English comprehension exam source documents.

Each markdown file contains:

- an improved learner-friendly passage
- 20 comprehension questions
- four answer choices per question
- answer key
- short explanation for review mode
- light spelling/vocabulary checks woven naturally into comprehension
